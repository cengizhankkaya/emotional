# Flutter Projesinde Image ve Asset Yönetimi

Bu projede asset ve image yönetimi için **flutter_gen** paketi kullanılmıştır. Bu yaklaşım, asset yollarını (string path) manuel olarak yazmak yerine, otomatik oluşturulan (generated) sınıflar üzerinden type-safe (tür güvenli) bir şekilde erişim sağlar.

## 1. Paketlerin Eklenmesi ve Yapılandırma

`pubspec.yaml` dosyasının ilgili bölümlerinin tamamı aşağıdadır.

### Kodun Tamamı: `pubspec.yaml` (Asset ile ilgili kısımlar)

```yaml
dependencies:
  flutter:
    sdk: flutter
  
  # SVG ve Lottie desteği için
  flutter_svg: ^2.0.9
  lottie: ^3.1.2

dev_dependencies:
  build_runner: ^2.3.3
  flutter_gen_runner: ^5.2.0

# Dosyanın en altındaki flutter_gen yapılandırması
flutter_gen:
  output: lib/product/generated/
  integrations:
    lottie: true
    flutter_svg: true

# Flutter asset tanımları
flutter:
  uses-material-design: true

  assets:
    - assets/translations/
    - assets/icons/
    - assets/docs/
    - assets/lottie/
    - assets/images/
    - assets/svg/
```

## 2. Kullanım Örnekleri (Kodların Tamamı)

Asset yapısının kullanıldığı gerçek dosya örnekleri aşağıdadır.

### Örnek 1: Görsel Kullanımı (`SplashView`)

`lib/features/splash/splash_view.dart` dosyasında `Assets.icons.icApp.image()` ve `Assets.lottie.loadingGray.lottie()` kullanılmıştır.

```dart
import 'package:easy_localization/easy_localization.dart';
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:kartal/kartal.dart';
import 'package:lifeclient/product/generated/assets.gen.dart'; // Generated dosya
import 'package:lifeclient/product/init/language/locale_keys.g.dart';

class SplashView extends ConsumerStatefulWidget {
  const SplashView({super.key});

  @override
  ConsumerState<ConsumerStatefulWidget> createState() => _SplashViewState();
}

class _SplashViewState extends ConsumerState<SplashView> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            // Generated Image kullanımı
            Assets.icons.icApp.image(
              height: context.sized.dynamicHeight(.3),
            ),
            Column(
              children: [
                Text("PROJECT NAME"),
                Padding(
                  padding: const EdgeInsets.only(top: 20),
                  // Generated Lottie kullanımı
                  child: Assets.lottie.loadingGray.lottie(),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}
```

### Örnek 2: PDF Dosyası Kullanımı (`KvkkCheckBox`)

Bu örnekte `Assets.docs.kvkk` string path olarak `SfPdfViewer.asset` içine verilmiştir. `assets.gen.dart` dosyasını `package:lifeclient/product/generated/assets.gen.dart` olarak import ediyoruz.

```dart
import 'package:flutter/cupertino.dart';
import 'package:flutter/material.dart';
import 'package:lifeclient/product/generated/assets.gen.dart';
import 'package:syncfusion_flutter_pdfviewer/pdfviewer.dart';

class KvkkCheckBox extends StatelessWidget {
  const KvkkCheckBox({super.key});

  void _navigate(BuildContext context) {
    Navigator.push(
      context,
      CupertinoPageRoute<void>(
        builder: (BuildContext context) {
          return Scaffold(
            appBar: AppBar(title: Text("KVKK")),
            // Asset path'ini string olarak alma: Assets.docs.kvkk
            body: SfPdfViewer.asset(Assets.docs.kvkk),
          );
        },
      ),
    );
  }

  @override
  Widget build(BuildContext context) {
    return CheckboxListTile(
      value: false, 
      onChanged: (val) => _navigate(context),
      title: Text("KVKK Metnini Oku"),
    );
  }
}
```

## 3. Kodun Oluşturulması

Yapılandırmayı yaptıktan sonra terminalde şu komutu çalıştırmanız yeterlidir:

```bash
dart run build_runner build --delete-conflicting-outputs
```
