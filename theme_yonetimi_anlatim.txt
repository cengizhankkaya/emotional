# Proje Tema ve Renk Yönetim Yapısı

Bu projede tema yönetimi, Flutter'ın standart `ThemeData` sınıfı üzerine kurulu özelleştirilmiş bir yapı ile sağlanmaktadır. Renkler tek bir noktadan yönetilir (`ColorsCustom`), tema tanımları ise `ApplicationTheme` sınıfı içinde yapılır ve `main.dart` (App) tarafında uygulamaya enjekte edilir.

Bu yapıyı kendi projenize entegre etmek için aşağıdaki adımları izleyebilirsiniz.

## 1. Renklerin Tanımlanması (ColorsCustom)

Tüm özel renkler `lib/product/utility/decorations/colors_custom.dart` dosyasında `static const` olarak tanımlanır. Bu, renk kodlarını kodun içine dağıtmak yerine tek bir yerden yönetmeyi sağlar.

### Kodun Tamamı: `lib/product/utility/decorations/colors_custom.dart`

```dart
import 'package:flutter/material.dart';

@immutable
final class ColorsCustom {
  const ColorsCustom._();
  static const Color sambacus = Color(0xFF101A2A);
  static const Color endless = Color(0xFFE23E3E);
  static const Color braziliante = Color(0xFF34C759);
  static const Color brandeisBlue = Color(0xFF0D6EFD);
  static const Color lightGray = Color(0xffCED4DA);
  static const Color softGray = Color.fromRGBO(219, 219, 219, 1);
  static const Color gray = Color.fromRGBO(245, 245, 245, 1);
  static const Color warmGrey = Color.fromRGBO(153, 153, 153, 1);
  static const Color darkGray = Color.fromRGBO(91, 91, 91, 1);
  static const Color underlinePurple = Color.fromRGBO(82, 0, 255, 1);
  static const Color white = Color(0xFFFFffff);
  static const Color imperilRead = Color(0xffEF2636);
  static const Color royalPeacock = Color(0xff27AAE5);
  static const Color transparent = Color(0x00000000);
  static const Color black = Color(0xFF000000);
  static const Color green = Color(0xFF00FF00);
}
```

## 2. Temanın Oluşturulması (ApplicationTheme)

Tema mantığı `lib/product/init/application_theme.dart` dosyasında bulunur. Burada `ThemeData` özelleştirilir ve `ColorsCustom` içindeki renkler kullanılarak genel uygulama stili belirlenir.

### Kodun Tamamı: `lib/product/init/application_theme.dart`

```dart
import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';
import 'package:kartal/kartal.dart'; // Projenizde kartal paketi yoksa bu importu kaldırın.
import 'package:lifeclient/product/utility/constants/index.dart'; // AppConstants kullanımı için
import 'package:lifeclient/product/utility/decorations/colors_custom.dart';
import 'package:lifeclient/product/utility/decorations/custom_radius.dart'; // CustomRadius kullanımı için

final class ApplicationTheme {
  ApplicationTheme.build(BuildContext context) {
    // final textTheme = context.general.textTheme; // Kartal kullanımı
    final theme = ThemeData.light(useMaterial3: true);
    final textTheme = theme.textTheme;
    themeData = theme.copyWith(
      timePickerTheme: const TimePickerThemeData(
        hourMinuteTextColor: Colors.black,
        hourMinuteColor: ColorsCustom.softGray,
        dayPeriodColor: Colors.black,
        dialHandColor: Colors.black,
      ),
      dialogTheme: const DialogThemeData(
        backgroundColor: ColorsCustom.white,
      ),
      bottomSheetTheme: const BottomSheetThemeData(
        backgroundColor: ColorsCustom.white,
      ),
      scaffoldBackgroundColor: ColorsCustom.white,
      floatingActionButtonTheme: const FloatingActionButtonThemeData(
        foregroundColor: ColorsCustom.sambacus,
        backgroundColor: ColorsCustom.white,
      ),
      iconButtonTheme: IconButtonThemeData(
        style: IconButton.styleFrom(
          foregroundColor: ColorsCustom.sambacus,
        ),
      ),
      cardTheme: const CardThemeData(
        elevation: 2,
        color: ColorsCustom.white,
        shape: RoundedRectangleBorder(
          borderRadius: BorderRadius.all(Radius.circular(12)), // CustomRadius.medium yerine
        ),
      ),
      appBarTheme: const AppBarTheme(
        scrolledUnderElevation: 0,
        foregroundColor: ColorsCustom.white,
        backgroundColor: ColorsCustom.white,
        titleTextStyle: TextStyle(
          color: ColorsCustom.sambacus,
          fontSize: 16,
          fontWeight: FontWeight.w600,
        ),
        iconTheme: IconThemeData(
          color: ColorsCustom.sambacus,
        ),
      ),
      textTheme: GoogleFonts.robotoTextTheme(textTheme).copyWith(),
      colorScheme: theme.colorScheme.copyWith(
        primary: ColorsCustom.sambacus,
        secondary: ColorsCustom.white,
        onPrimaryContainer: ColorsCustom.lightGray,
        onPrimaryFixed: ColorsCustom.gray,
        error: ColorsCustom.imperilRead,
        primaryContainer: ColorsCustom.braziliante,
        onTertiaryContainer: ColorsCustom.green,
        onSecondaryContainer: ColorsCustom.royalPeacock,
        onSecondaryFixed: ColorsCustom.warmGrey,
        onPrimaryFixedVariant: ColorsCustom.darkGray,
        onTertiaryFixedVariant: ColorsCustom.underlinePurple,
      ),
      listTileTheme: const ListTileThemeData(
        titleTextStyle: TextStyle(
          color: Colors.black,
          fontSize: 16,
          fontWeight: FontWeight.w600,
        ),
      ),
      progressIndicatorTheme: const ProgressIndicatorThemeData(
        color: Colors.black,
      ),
      snackBarTheme: SnackBarThemeData(
        backgroundColor: ColorsCustom.black,
        // context.general.textTheme kullanımı için context gerekir veya standart style verilebilir
        contentTextStyle: const TextStyle(color: ColorsCustom.white), 
      ),
      scrollbarTheme: ScrollbarThemeData(
        thickness: WidgetStatePropertyAll(1.0), // AppConstants.kOne yerine
      ),
    );
  }
  late final ThemeData themeData;

  static double maxWeight = 1000;
}
```

## 3. Yardımcı Sınıf (ColorCommon)

### Kodun Tamamı: `lib/product/common/color_common.dart`

```dart
import 'package:flutter/material.dart';
import 'package:kartal/kartal.dart'; // Kartal paketi extensionları
import 'package:lifeclient/product/utility/decorations/colors_custom.dart';

/// more detail for look at
/// https://m3.material.io/styles/color/the-color-system/tokens
final class ColorCommon {
  ColorCommon(this.context);

  final BuildContext context;

  static const Color sameWhiteColor = ColorsCustom.white;
  static const Color linkColor = ColorsCustom.brandeisBlue;

  /// means [ThemeData].light == white
  Color get whiteAndBlackForTheme {
    return context.general.colorScheme.onSurface;
  }

  /// means [ThemeData].light == black
  Color get blackAndWhiteForTheme {
    return context.general.colorScheme.surface;
  }
}
```

## 4. Temanın Uygulamaya Bağlanması

`main.dart` veya `app.dart` dosyasında kullanımı:

```dart
// lib/features/app.dart (Özet)
Widget build(BuildContext context, WidgetRef ref) {
  return MaterialApp.router(
    // ...
    theme: ApplicationTheme.build(context).themeData, 
    // ...
  );
}
```
