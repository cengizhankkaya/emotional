Flutter Projesi Başlangıç Yapısı (Application Initialization Pattern)

Bu yapı, "Separation of Concerns" (İlgi Alanlarının Ayrımı) prensibini uygulayarak main.dart dosyasını temiz tutmayı ve uygulama başlamadan önceki tüm hazırlıkları tek bir yerden yönetmeyi sağlar.

### 1. Dosya Klasör Yapısı
Projenizde aşağıdaki klasör ve dosya yapısını oluşturun:
- lib/
  - core/
    - init/
      - core_localize.dart
  - product/
    - init/
      - application_init.dart
  - main.dart

---

### 2. Dil Ayarları (CoreLocalize)
Dil ayarlarını yöneten basit bir sınıf.

Dosya: lib/core/init/core_localize.dart

```dart
import 'package:flutter/material.dart';

// Desteklenen dilleri burada enum olarak tanımlayın
enum AppLocale {
  en(Locale('en', 'US')),
  tr(Locale('tr', 'TR'));

  const AppLocale(this.locale);
  final Locale locale;
}

@immutable
final class CoreLocalize {
  // Çeviri dosyalarının (json) duracağı klasör yolu
  // assets/translations klasörünün pubspec.yaml'da tanımlı olduğundan emin olun
  final String initialPath = 'assets/translations';
  
  // Uygulama açılışındaki varsayılan dil
  final Locale startLocale = AppLocale.tr.locale;
  
  // Desteklenen tüm dillerin listesi
  final List<Locale> supportedItems =
      AppLocale.values.map((e) => e.locale).toList();
}
```

---

### 3. Başlatıcı Sınıf (ApplicationInit)
Uygulama başlamadan önce çalışması gereken tüm servisleri (Firebase, Cache, vb.) başlatan sınıf.

Dosya: lib/product/init/application_init.dart

```dart
import 'dart:async';
import 'package:easy_localization/easy_localization.dart';
import 'package:flutter/material.dart';
import 'package:flutter/services.dart';
// Gerekirse diğer paketleri import edin (örn: firebase_core)
import '../../core/init/core_localize.dart';

@immutable
final class ApplicationInit {
  // Yukarıda yazdığımız dil sınıfını buraya bağlıyoruz
  final CoreLocalize localize = CoreLocalize();

  Future<void> start() async {
    // 1. Flutter Motorunu Bağla (Zorunlu)
    WidgetsFlutterBinding.ensureInitialized();
    
    // 2. Dil Paketini Başlat
    await EasyLocalization.ensureInitialized();

    // 3. Ekran Yönünü Kilitle (Opsiyonel - Sadece dikey mod)
    await _setRotation();

    // 4. Firebase Başlat (Eğer kullanıyorsanız yorum satırını kaldırın)
    // await Firebase.initializeApp();

    // 5. Diğer servisleri başlat (Cache, Singleton'lar vb.)
    // await ProjectDependency.setup(); 
  }

  // Yardımcı metod: Ekranı dikey moda sabitler
  Future<void> _setRotation() async {
    await SystemChrome.setPreferredOrientations([
      DeviceOrientation.portraitUp,
      DeviceOrientation.portraitDown,
    ]);
  }
}
```

---

### 4. Main Dosyası (main.dart)
Artık main.dart dosyanız sadece başlatıcıyı çağırır ve uygulamayı ayağa kaldırır.

Dosya: lib/main.dart

```dart
import 'package:easy_localization/easy_localization.dart';
import 'package:flutter/material.dart';
import 'product/init/application_init.dart'; // Yol paket ismine göre değişebilir

void main() async {
  // Başlatıcı sınıfı oluştur
  final initialManager = ApplicationInit();
  
  // Her şeyi başlat ve bitmesini bekle
  await initialManager.start();

  runApp(
    // EasyLocalization'ı en tepede sarmalıyoruz
    EasyLocalization(
      supportedLocales: initialManager.localize.supportedItems,
      path: initialManager.localize.initialPath,
      startLocale: initialManager.localize.startLocale,
      useOnlyLangCode: true,
      child: const MyApp(), // Sizin ana widget'ınız
    ),
  );
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Örnek Proje',
      // Dil ayarlarını EasyLocalization context'inden al
      localizationsDelegates: context.localizationDelegates,
      supportedLocales: context.supportedLocales,
      locale: context.locale, // Dil değiştiğinde tetiklenir
      home: const Scaffold(
        body: Center(
          child: Text("Uygulama Başladı"),
        ),
      ),
    );
  }
}
```

### Özet Avantajlar
1. main() fonksiyonu temiz kalır.
2. Yeni bir servis eklemek (örn. Push Notification) için sadece ApplicationInit sınıfına ekleme yaparsınız.
3. Dil ve konfigürasyon ayarları merkezi bir yerden yönetilir.
