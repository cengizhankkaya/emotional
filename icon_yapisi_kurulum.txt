# Flutter Projesinde Icon Yapısı Kurulumu

Bu dokümanda, Flutter projesinde icon yönetimi için gerekli paketlerin kurulumu ve kullanımı anlatılmaktadır.

## 1. Paketlerin Eklenmesi

`pubspec.yaml` dosyasına aşağıdaki paketleri ekleyin:

```yaml
dependencies:
  flutter:
    sdk: flutter
  
  # Icon Paketleri
  font_awesome_flutter: ^10.5.0
  hugeicons: ^0.0.7

dev_dependencies:
  flutter_launcher_icons: ^0.14.1
```

## 2. Launcher Icon (Uygulama İkonu) Yapılandırması

### 2.1. Icon Dosyasını Hazırlama

1. Uygulama ikonunuzu hazırlayın (minimum 1024x1024 px, PNG formatında)
2. Projenizde `assets/app/` klasörü oluşturun
3. Icon dosyasını `assets/app/app_icon.png` olarak kaydedin

### 2.2. pubspec.yaml Yapılandırması

`pubspec.yaml` dosyasının en altına ekleyin:

```yaml
flutter_launcher_icons:
  android: "launcher_icon"
  ios: true
  image_path: "assets/app/app_icon.png"
  min_sdk_android: 21
  remove_alpha_ios: true
```

### 2.3. Icon Oluşturma

Terminalde şu komutu çalıştırın:

```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

Bu komut, Android ve iOS için tüm boyutlarda launcher icon'ları otomatik oluşturur.

## 3. Merkezi Icon Yönetimi (AppIcons Sınıfı)

### 3.1. Dosya Oluşturma

`lib/product/utility/constants/app_icons.dart` dosyasını oluşturun.

### 3.2. Kodun Tamamı

```dart
import 'package:flutter/material.dart';
import 'package:font_awesome_flutter/font_awesome_flutter.dart';

/// Uygulama genelinde kullanılan icon'ların merkezi yönetimi
final class AppIcons {
  AppIcons._();

  // Genel İşlemler
  static const IconData close = Icons.close_outlined;
  static const IconData info = Icons.info_outlined;
  static const IconData add = Icons.add_outlined;
  static const IconData delete = Icons.delete_outlined;
  static const IconData edit = Icons.edit_outlined;
  static const IconData save = Icons.save_outlined;
  
  // Navigasyon
  static const IconData home = Icons.home;
  static const IconData search = Icons.search_outlined;
  static const IconData settings = Icons.settings_outlined;
  static const IconData back = Icons.arrow_back;
  static const IconData forward = Icons.arrow_forward;
  static const IconData menu = Icons.menu_outlined;
  
  // İletişim
  static const IconData phone = Icons.phone;
  static const IconData email = Icons.email_outlined;
  static const IconData share = Icons.share_rounded;
  static const IconData sendMessage = FontAwesomeIcons.whatsapp;
  
  // Konum
  static const IconData location = Icons.location_on;
  static const IconData map = Icons.map_outlined;
  
  // Medya
  static const IconData camera = Icons.camera_alt_outlined;
  static const IconData gallery = Icons.image_outlined;
  static const IconData photo = Icons.photo_outlined;
  static const IconData video = Icons.videocam_outlined;
  
  // Zaman
  static const IconData calendar = Icons.calendar_today;
  static const IconData time = Icons.access_time;
  
  // Kullanıcı
  static const IconData person = Icons.person_outlined;
  static const IconData group = Icons.group_outlined;
  
  // Favoriler
  static const IconData favorite = Icons.favorite;
  static const IconData favoriteBorder = Icons.favorite_border_outlined;
  
  // Bildirimler
  static const IconData notifications = Icons.notifications_outlined;
  static const IconData notificationsOff = Icons.notifications_off_outlined;
  
  // Diğer
  static const IconData filter = Icons.filter_list_outlined;
  static const IconData sort = Icons.sort_outlined;
  static const IconData download = Icons.download_outlined;
  static const IconData upload = Icons.upload_outlined;
}
```

## 4. Kullanım Örnekleri

### 4.1. Material Icons (AppIcons üzerinden)

```dart
// Icon widget ile
Icon(AppIcons.home)
Icon(AppIcons.settings, color: Colors.blue, size: 24)

// IconButton ile
IconButton(
  icon: Icon(AppIcons.notifications),
  onPressed: () {
    // Bildirimler sayfasına git
  },
)

// ListTile ile
ListTile(
  leading: Icon(AppIcons.person),
  title: Text('Profil'),
  trailing: Icon(AppIcons.forward),
)
```

### 4.2. Font Awesome Icons

```dart
// WhatsApp icon
Icon(AppIcons.sendMessage)

// Direkt kullanım
Icon(FontAwesomeIcons.facebook)
Icon(FontAwesomeIcons.twitter)
Icon(FontAwesomeIcons.instagram)
```

### 4.3. HugeIcons

```dart
import 'package:hugeicons/hugeicons.dart';

HugeIcon(
  icon: HugeIcons.strokeRoundedArrowDown01,
  color: Colors.blue,
  size: 24,
)
```

### 4.4. Direkt Material Icons

```dart
Icon(Icons.add)
Icon(Icons.delete_outline)
Icon(Icons.edit_outlined)
```

## 5. Asset Olarak Custom Icon Kullanımı

### 5.1. Klasör Yapısı

```
assets/
  ├── icons/
  │   ├── custom_icon_1.png
  │   └── custom_icon_2.png
  └── svg/
      ├── logo.svg
      └── icon.svg
```

### 5.2. pubspec.yaml'a Ekleme

```yaml
flutter:
  assets:
    - assets/icons/
    - assets/svg/
```

### 5.3. Kullanım

```dart
// PNG icon
Image.asset(
  'assets/icons/custom_icon_1.png',
  width: 24,
  height: 24,
)

// SVG icon (flutter_svg paketi gerekli)
import 'package:flutter_svg/flutter_svg.dart';

SvgPicture.asset(
  'assets/svg/logo.svg',
  width: 100,
  height: 100,
)
```

## 6. Script Oluşturma (Opsiyonel)

Icon'u güncellemek için script oluşturabilirsiniz.

### 6.1. scripts/update_icon.sh

```bash
#!/bin/bash

echo "🎨 Launcher icon güncelleniyor..."
flutter pub run flutter_launcher_icons
echo "✅ Icon güncelleme tamamlandı!"
```

### 6.2. pubspec.yaml'a Script Ekleme

```yaml
scripts:
  updateIcon: sh ./scripts/update_icon.sh
```

### 6.3. Kullanım

```bash
# Script'i çalıştırılabilir yapma
chmod +x scripts/update_icon.sh

# Çalıştırma
sh scripts/update_icon.sh
```

## 7. Best Practices (En İyi Uygulamalar)

1. **Merkezi Yönetim**: Tüm icon'ları `AppIcons` sınıfında toplayın
2. **Anlamlı İsimler**: Icon isimlerini açıklayıcı yapın (örn: `sendMessage` yerine `whatsapp`)
3. **Outlined Tercih Edin**: Material Icons'da outlined versiyonları tercih edin (daha modern görünüm)
4. **Boyut Standardı**: Icon boyutlarını 24px standart olarak kullanın
5. **Renk Yönetimi**: Icon renklerini tema üzerinden yönetin
6. **SVG Kullanın**: Custom icon'lar için PNG yerine SVG tercih edin (ölçeklenebilir)

```

```

## 10. Özet

✅ **Material Icons**: Flutter'ın yerleşik icon kütüphanesi (ücretsiz, 1000+ icon)
✅ **Font Awesome**: Sosyal medya ve marka icon'ları (ücretsiz, 2000+ icon)
✅ **HugeIcons**: Modern ve özel icon'lar (ücretli/ücretsiz seçenekler)
✅ **flutter_launcher_icons**: Uygulama ikonu otomatik oluşturma
✅ **AppIcons sınıfı**: Merkezi icon yönetimi için best practice

Bu yapı sayesinde icon'larınızı merkezi bir yerden yönetebilir, kolayca değiştirebilir ve projenizde tutarlılık sağlayabilirsiniz.
