RESPONSIVE YAPI VE EKRAN BOYUTLANDIRMA KURULUMU

Bu rehber, projedeki "ResponsiveBreakpoints" yapısını ve "TextScaler" (yazı boyutu sabitleme) ayarını başka bir projeye nasıl aktaracağınızı adım adım anlatır.

ADIM 1: Gerekli Paketin Eklenmesi
---------------------------------------------------------
Öncelikle `pubspec.yaml` dosyanıza `responsive_framework` paketini ekleyin.
(Versiyon numarası projeye göre değişebilir, güncel olanı kullanmanız önerilir ancak bu projedeki sürüm aşağıdadır.)

dependencies:
  flutter:
    sdk: flutter
  responsive_framework: ^1.1.0

Terminalden çalıştırmak için:
flutter pub add responsive_framework


ADIM 2: AppBuilder Yardımcı Sınıfını Oluşturma
---------------------------------------------------------
Projenizde düzenli bir yapı olması için `lib/product/` veya `lib/core/init/` gibi bir klasör altına `app_builder.dart` adında bir dosya oluşturun ve şu kodları yapıştırın:

import 'package:flutter/material.dart';
import 'package:responsive_framework/responsive_framework.dart';

class AppBuilder {
  AppBuilder._();

  static Widget build(BuildContext context, Widget? child) {
    return MediaQuery(
      // TextScaler.noScaling: Kullanıcının telefon ayarlarındaki yazı büyütme
      // tercihini yoksayar ve uygulamanın her zaman standart boyutta görünmesini sağlar.
      // Eğer erişilebilirlik (büyük yazı) desteği vermek isterseniz bu satırı kaldırabilirsiniz.
      data: MediaQuery.of(context).copyWith(textScaler: TextScaler.noScaling),
      child: ResponsiveBreakpoints.builder(
        child: child ?? const SizedBox(),
        breakpoints: [
          const Breakpoint(start: 0, end: 450, name: MOBILE),
          const Breakpoint(start: 451, end: 800, name: TABLET),
          const Breakpoint(start: 801, end: 1920, name: DESKTOP),
          const Breakpoint(start: 1921, end: double.infinity, name: '4K'),
        ],
      ),
    );
  }
}


ADIM 3: Main veya App Dosyasına Entegre Etme
---------------------------------------------------------
Uygulamanızın ana giriş noktası olan `MaterialApp` (veya `MaterialApp.router`) widget'ının `builder` parametresine bu yapıyı bağlamanız gerekmektedir.

Örnek `main.dart` veya `app.dart`:

import 'package:flutter/material.dart';
import 'package:example_project/product/app_builder.dart'; // Yeni oluşturduğunuz dosya yolu

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Örnek Proje',
      debugShowCheckedModeBanner: false,
      
      // KRİTİK NOKTA: builder parametresini AppBuilder.build'e bağlıyoruz.
      builder: AppBuilder.build,
      
      home: const Scaffold(
        body: Center(child: Text("Merhaba Responsive Dünya")),
      ),
    );
  }
}

---------------------------------------------------------
ÖZET:
1. `responsive_framework` paketini ekle.
2. `AppBuilder` sınıfını oluştur ve breakpoint'leri tanımla.
3. `MaterialApp` -> `builder` özelliğine `AppBuilder.build` metodunu ver.
